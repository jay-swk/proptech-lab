# Plan: 개념 퀴즈 — 인터랙티브 학습 검증

> CPS Framework | 2026-04-03 | Phase: Plan

---

## Context

- 개념 사전에 6개 개념이 있지만, 읽기만 하면 학습 효과가 약함
- "읽고 → 풀고 → 시뮬레이터에서 확인" 학습 루프를 만들면 정착률 향상
- 퀴즈는 SSG로 정적 생성 가능 (서버 불필요)

## Problem

### P1. 수동적 학습
- 현재는 읽기만 가능. 이해도 확인 방법이 없음.

### P2. 시뮬레이터와의 연결 부재
- 개념을 배운 후 "직접 해보기"로 연결되는 경험이 없음

## Solution

### S1. 개념별 퀴즈 (4지선다, 3문제씩)

각 개념 상세 페이지 하단에 퀴즈 섹션 추가.

**퀴즈 데이터 구조**:
```typescript
interface Quiz {
  question: string;
  options: string[];
  correctIndex: number;
  explanation: string;
  simulatorLink?: { label: string; params: Partial<SimulatorParams> };
}
```

**퀴즈 목록** (개념당 3문제):

용적률 퀴즈:
1. "330m² 대지에 용적률 200%가 적용되면 최대 연면적은?" → 660m²
2. "용적률이 높을수록 어떻게 되나?" → 더 높은 건물을 지을 수 있다
3. "제2종 일반주거지역의 용적률 상한은?" → 250%

건폐율 퀴즈:
1. "건폐율이 제한하는 것은?" → 건물 바닥면적
2. "500m² 대지에 건폐율 60%면 최대 건축면적은?" → 300m²
3. "건폐율이 낮으면 어떤 점이 좋은가?" → 일조/통풍/녹지 확보

(나머지 4개 개념도 각 3문제씩)

**동작**:
1. 개념 상세 페이지 하단에 "이해도 확인" 섹션
2. 한 문제씩 표시, 답 선택 시 즉시 정답/오답 피드백
3. 오답 시 설명 + (있으면) "시뮬레이터에서 직접 확인" 링크
4. 모든 문제 완료 시 점수 표시 + "시뮬레이터로 가기" CTA
